Moodang Odyssey

다른 차들을 피하고 2개의 정류장에서 미니게임을 통과해 등교 완료하는 게임

Duration

4weeks (2023.10.14 ~ 2023.11.17)

Tech Stack

HTML, CSS, JavaScript, Three.js, React

Description

 Main Game

다른 차들과 부딪히지 않고 총 3개의 정류장에 도달하고 Moodang(무당이)을 등교 완료시킨다.

 Mini Game 1 (무당이를 탈출시켜라!)

보드게임 'Rush Hour'에서 착안한 게임. 다른 차들을 움직여 Moodang(무당이)를 exit으로 탈출시킨다.

 Mini Game 2 (업무 피하기 대작전)

Moodang(무당이)가 술래의 눈을 피해 박스로 몸을 숨기고 탈출해야 한다.
