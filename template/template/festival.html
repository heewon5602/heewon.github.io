Festival Finder

개최 예정 또는 개최 중인 지역 축제를 보여주는 웹 페이지

Duration

2weeks (2023.11.06 ~ 2023.11.20)

Tech Stack

Amazon Route 53, Amazon S3, CSS, HTML, CSS, Vanilla JS, json, AWS IAM, Docker, MySQL, Open API, CI/CD

Description

Main Page

모든 지역의 개최 예정 또는 진행 중인 축제들을 보여준다.

Region Page (Sub Page)

List에서 지역 선택 시 해당 지역 관련 축제들만 보여준다.

Detail Page

해당 지역에 대한 여러 정보 (개최여부, 상세 설명, 개최일, 개최 마감일, category, 지역, 장소, performer)를 보여준다.
